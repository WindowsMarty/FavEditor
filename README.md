# FavEditor
FavEditor，替换Safari主页丑陋的书签图标。基于Favman项目二次开发。

## 功能改进
1. 直接读取本地书签列表，而非缓存
2. 导入图标时，应用将进行自动裁切
3. 其他一些功能和UI改进

## 使用注意

1. 当首次进入应用时，需要在左侧菜单栏批准权限，应用才可以访问Safari本地文件夹。
2. 更改书签的图标后，需要在应用内及时锁定，否则Safari可能会在下一次访问该网站后刷新图标缓存
3. 可直接从网页中拖动图标至应用内相应区域来快速更换
